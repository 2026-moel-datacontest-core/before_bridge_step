# 각 디렉터리 및 파일의 역할

## 목적

이 문서는 `law_main_road` 저장소를 처음 보는 사람이 어디에 무엇이 있는지 빠르게 파악할 수 있도록 만든 저장소 지도다.

이 문서는 모든 파일을 하나씩 설명하지 않는다. 대신 아래 기준으로 정리한다.

- 실제 실행에 중요한 디렉터리와 엔트리 파일
- 자주 수정하는 소스 파일
- 생성물 / 캐시 / 로그처럼 필요할 때만 보면 되는 경로
- 운영 문서와 작업 기록 경로
