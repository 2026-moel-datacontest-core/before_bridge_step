# Recovery Flow

기준일: `2026-04-20`

## 현재 상태

Recovery는 제품 확장 단계이며 현재 MVP / frontend demo 범위가 아니다.

현재 기준:

- Recovery route 없음
- 실제 제출 기능 없음
- PDF / HWP export 없음
- 사용자 계정 / 서버 저장 없음
- 2026-04-20 기준 presentation-local preset과 SCN-004 document draft는 After flow 보조 기능이며 Recovery 본 구현이 아님

## 제품상 목표

Recovery는 사용자가 법령 근거와 사건 정리를 확인한 뒤, 후속 회복이나 전환을 준비하도록 돕는 단계다.

후속 후보:

- 기관 제출 전 체크리스트
- 상담용 사건 요약서
- 신청/진정/이유서 문서 초안 고도화
- 제출 후 일정 관리
- 지원기관 안내

## 현재 구현과의 관계

SCN-004 document draft는 Recovery 전체 구현이 아니라 After 단계의 제출 전 검토용 초안 보조 기능이다. 따라서 현재 구현을 Recovery 본격 구현으로 설명하지 않는다.

## 후속 조건

Recovery는 SCN-004 demo freeze 기준을 유지한 뒤 별도 scope로 검토한다.
